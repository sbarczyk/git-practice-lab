# Raport z laboratorium — GIT

Repozytorium zawiera raport i wykonane ćwiczenia z systemu kontroli wersji **Git**, realizowane w ramach zajęć na **Akademii Górniczo-Hutniczej w Krakowie (Wydział Informatyki)**.  
Zadania pochodzą z platformy [gitexercises.fracz.com](https://gitexercises.fracz.com), stworzonej przez dr. inż. Krzysztofa Frączka, i obejmują praktyczne scenariusze pracy z repozytoriami Git.

Celem projektu było zrozumienie i przećwiczenie podstawowych oraz średniozaawansowanych mechanizmów Gita, takich jak:
- tworzenie i łączenie commitów,  
- praca z branchami i tagami,  
- cofanie zmian i rozwiązywanie konfliktów,  
- współpraca w zdalnym repozytorium.
